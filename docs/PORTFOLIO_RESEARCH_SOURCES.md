## Methodology Notes

### Research Standards

**Tier-1 Sources Prioritized:**
- Gartner, Forrester, IDC, McKinsey, Deloitte
- Markets and Markets, Grand View Research, Mordor Intelligence
- University research institutions
- Public company financial reports

**Data Triangulation:**
- Multiple sources confirm key data points
- Conservative estimates used when range exists
- Market size data cross-referenced across 3+ sources

**Citation Format:**
- Source name, report title, URL (where public)
- Key data point extracted
- Date/year of publication
- Access date: January 2026

### Confidence Levels

**High Confidence (90%+):**
- Data from tier-1 analyst firms
- Multiple confirming sources
- Recent publications (2024-2025)

**Medium Confidence (70-90%):**
- Single tier-1 source
- Slightly dated data (2022-2023)
- Industry estimates vs hard data

**Lower Confidence (<70%):**
- Emerging markets with limited data
- Projections beyond 5 years
- Niche segments
