Differentiation strengths

κ (kappa) invariant — no competitor implements this. Detects circular dependencies in knowledge graphs using Tarjan's SCC algorithm. Routes agent inference: κ=0 fast single-pass; κ>0 triggers deliberation with fault-line decomposition. Verified on 1.9M+ finite systems.

22 MCP tools — richest MCP API surface in the category. Hindsight has ~6 tool categories, Mem0 has a simpler REST/SDK, Zep is cloud-first. Graphonomous exposes graph mutation, typed retrieval, goal management, coverage queries, consolidation, topology analysis, and attention dispatch — all as MCP tools.

Goal management + attention engine — only Graphonomous treats goals as first-class graph citizens with epistemic coverage scoring (act/learn/escalate routing) and proactive attention dispatch (survey/triage/dispatch cycles). Zero competitors offer this.

11 interactive A/B demos — only 4.8% of SaaS companies offer interactive demos (Navattic 2025). Interactive demos drive 25% higher conversion, 70% more trial signups, and 7.2× more engagement than video demos. No direct competitor has an interactive demo suite — Mem0, Zep, Letta rely on docs/API playgrounds. Cognee has a playground in development.

Elixir/OTP backbone — built for concurrency and fault-tolerance where competitors use Python/TypeScript. OTP supervision trees provide self-healing runtime.

Edge-native — SQLite + sqlite-vec runs on a Raspberry Pi. Hindsight requires Docker, Zep requires Neo4j, Mem0 paywalls graph memory behind $249/mo.

Outcome learning with causal attribution — explicit causal_parent_ids linking outcomes back to beliefs. Confidence updates propagate through the causal chain. No competitor has this granularity.

Gaps to close

Community size — Mem0 (41K stars), Cognee (12K stars), Letta (~12–15K stars), Hindsight (3.8K stars) all have larger communities. Graphonomous is pre-traction. Registry listings (Smithery, MCP.so, PulseMCP, Glama, mcp-get) would change this quickly.

LongMemEval benchmark — CLOSED — Graphonomous achieves 90.4% Session Hit Rate on LongMemEval (neural, 100 questions), within 1pp of Hindsight’s 91.4% SOTA. Beats Zep (~63–67%), Letta (~50–80%), and GPT-4 128K (~62–65%). Full results in OS-E001 §4.5.

Revenue signal — every funded competitor has either a paid tier or clear monetization path. Mem0 from $19/mo (graph at $249/mo), Zep from $25/mo. A managed cloud or pro tier is the next investor milestone.

LLM provider breadth — Hindsight supports OpenAI, Anthropic, Gemini, Groq, Ollama out of box. Graphonomous uses local embeddings (all-MiniLM-L6-v2) which is an advantage for edge but limits cloud flexibility.

Phase 3 deferred — Postgres adapter, Phoenix dashboard, and federation are designed but not implemented.
